How to Use

1

Download & Run

Download installer.exe and run it as administrator (right-click → Run as administrator)

2

Choose Installation Type

The installer will ask you:

  • Option A: Automatic - Detects ATLauncher in default locations
  • Option B: Manual - You specify where ATLauncher is installed

Common ATLauncher Locations

C:\Users\YourName\ATLauncher
C:\Program Files\ATLauncher
C:\Users\YourName\AppData\Roaming\ATLauncher
3

Wait for Installation

The installer will show progress:

  • Creating ATLauncher instance
  • Installing FrostClient files
  • Configuring Minecraft 1.8.9
  • Installing cape support
4

Launch ATLauncher

After installation completes:

  • Open ATLauncher
  • Wait 30 seconds to 2 minutes
  • Look for "FrostClient (Classic)" in instances
  • Click Play!
