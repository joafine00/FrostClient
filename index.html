All Mods & Modules

The FrostClient Classic includes all core Cloud Client modules, plus additional customizations:

  • Mod Categories
  • Search Bar
  • Nickhider Mod (textbox)
  • Crosshair Mod (with presets and delete button)
  • Scroll Zoom
  • Nametag Mod
  • Scoreboard Mod
  • Bossbar Mod
  • Direction HUD
  • Fireheight Option
  • Minimal View Bobbing
  • Colorpicker Rewrite
  • Additional FrostClient-exclusive modules (custom HUDs, QoL features)

Note: This list covers documented Cloud Client modules; FrostClient Classic may include additional proprietary modules.
